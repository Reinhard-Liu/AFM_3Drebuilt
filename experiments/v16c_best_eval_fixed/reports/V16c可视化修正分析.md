# V16c 可视化修正分析

## 修正内容

`src/visualize_val.py` 和 `src/visualize_5mol.py` 中，Predicted 3D 图此前使用 GT mask（`batch["atom_mask"]`）绘制、设置轴范围、以及计算 retrieval/similarity。修正后：
- 从 `n_atoms_pred` 构建 `pred_mask`：`pred_mask[:n_pred] = 1, 其余 = 0`
- Predicted 3D 图使用 `pred_mask` 绘制和设轴
- Retrieval 和 similarity 的 pred 侧使用 `pred_mask`
- GT 图继续使用 GT mask，两侧完全独立

## 修正前 vs 修正后对比

| 样本 | 旧版 Count | 新版 Count | 旧版 Sim | 新版 Sim | 变化原因 |
|------|-----------|-----------|---------|---------|---------|
| #0 | 26/27 Sim=0.522 | 26/27 Sim=0.516 | Count=1.000 | Count=0.963 | retrieval 用 pred_mask 后 count_sim 下降 |

**总结**：修正后的 Sim 值略低于旧版，因为旧版用 GT mask 计算 pred 侧，人为拉高了 count_sim 和 type_acc。差异在 2-3% 范围内，说明 GT mask 污染的影响存在但不算巨大。

## 修正后可视化的核心观察

### 1. 预测结构与 GT 的真实相似性

**结构展开正确但几何细节粗糙**：
- 所有 15 个样本中，预测结构**不再坍缩成一团**（V16b 的核心缺陷已完全消除）
- 预测结构的 XY 平面展开范围与 GT **大致一致**
- 但原子间距**明显小于 GT**：预测的原子聚集在一个更紧凑的区域内
- 典型案例：#0（GT 展开 XY ±0.5，Pred 展开 XY ±0.3），#713（GT Z 范围 ±0.5，Pred Z 范围 ±0.2）

**Z 轴压缩是最显著的几何缺陷**：
- 几乎所有样本中，GT 有清晰的 Z 方向展开（分子有上下层），Pred 的 Z 分布被压到接近平面
- 这直接导致了 Bottom Recall = 6% 的极低值
- #356 是唯一有一定 Z 展开的样本（Pred Z 达到 ±1.0），但它的整体 RMSD=0.445 反而更高

### 2. 原子类型可视化上的问题

**类型预测视觉上不可靠**：
- GT 图中可以清晰看到多种颜色（黑=C, 白=H, 蓝=N, 红=O, 黄=S 等）
- Pred 图中**几乎全部是灰色/白色小圆点**（空心圈），颜色区分度极低
- 具体表现：
  - #0：GT 有 CHNO（黑蓝红白），Pred 标签只有 CH
  - #499：GT 有 CHNO，Pred 标签只有 CH
  - #927：GT 有明显的黄色（S）和绿色（Cl），Pred 完全没有这些颜色
  - #356：GT 有蓝色（N），Pred 完全没有

- Type Acc 数值（0.38-0.71）看似不低，但可视化说明匹配准确率主要来自 C 和 H 的高占比（这两种占分子的 80%+），而 **N/O/S/F/Cl 等杂原子几乎不被预测到**

### 3. RMSD 不错但 3D 拓扑不像 GT 的样本

- **#0**（RMSD=0.303, Sim=0.516）：RMSD 合理，但 Pred 结构是一堆紧密的灰点，完全看不出 GT 那种清晰的芳香环骨架
- **#713**（RMSD=0.252, Sim=0.538）：同上，GT 有明确的平面芳香结构，Pred 是一团小点
- **#927**（RMSD=0.297, Sim=0.460）：GT 有黄色 S 和绿色 Cl 杂原子，Pred 完全没有
- **#356**（RMSD=0.445, Sim=0.455）：Pred 有几个原子跑到了 Z=1.5 的位置（异常离群点），导致高 RMSD

**结论：当前 RMSD 在 0.2-0.5 范围内不等于"结构相似"**。RMSD 主要反映的是匈牙利匹配后的平均距离，但分子的 3D 拓扑（环形、线性、分支）和原子类型分布与 GT 仍有根本差距。

### 4. 5-mol 对比图的观察

- **Pred 结构视觉上比 Top-1/2/3 retrieval 结果差很多**：
  - Top-1/2/3 都来自数据库真实分子，有正确的颜色、合理的结构展开
  - Pred 是一团灰白色小点，无法看出分子拓扑
- **Retrieval 排名不理想**：GT Rank 通常在 60-800+，说明预测结构在数据库中不能很好地对应到原始分子

## 需要撤回或修正的旧版结论

| 旧版结论 | 修正 |
|---------|------|
| "结构不再坍缩" | **仍然成立**。V16b 的一团坍缩确实消除了 |
| "原子数预测基本准确（±1-3）" | **仍然成立**。Count 指标和可视化一致 |
| "XY 平面投影基本合理" | **需修正**：XY 展开范围大致正确，但原子间距偏小，结构整体偏紧凑 |
| "Type Match 达到 40-70%" | **需修正**：数值主要由 C/H 高占比驱动，杂原子（N/O/S/F/Cl）类型预测几乎失败，可视化上看不到正确的杂原子颜色 |
| "部分样本类型准确率很高（如 #499 Type=0.71）" | **需撤回**：Type=0.71 主要因为 GT 中 C+H 占比高，模型把所有原子都预测成 C/H 就能得到高分，不代表真正学会了类型分辨 |
| "Z 轴仍然偏压缩" | **仍然成立，且比旧版分析更严重**：Z 压缩是普遍现象，不是个别样本 |
| "Bottom 原子几乎不可见" | **仍然成立** |

## 总结

修正 pred_mask 后，可视化结论整体比旧版更悲观：
1. **结构不再坍缩**是唯一可以确认的重大改善
2. **原子类型预测几乎只学会了 C 和 H**，杂原子基本缺失
3. **3D 拓扑（环形骨架、分支结构）仍未被学会**，预测结构只是"正确数量的原子在正确的大致位置"
4. RMSD 和 Sim 数值给出了过于乐观的印象，实际视觉质量远低于数值暗示
