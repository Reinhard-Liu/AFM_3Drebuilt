# 项目改进方案：基于 Video ViT 与条件化学扩散模型的 AFM 图像三维分子结构重建

---

## 一、项目背景与现存问题

本项目基于 Video Vision Transformer（Video ViT）编码器与条件去噪扩散概率模型（Conditional DDPM）实现从 AFM 图像堆栈到三维分子结构的端到端重建。根据反馈，前方案存在以下七个关键问题：

| 编号 | 问题描述 |
|------|---------|
| 1 | 生成的分子原子个数不正确：模型在推理时依赖真实原子数作为输入，无法自主预测 |
| 2 | 生成的分子物理结构不合理：原子间键长、键角不满足化学物理规律 |
| 3 | 无法识别分子 ID：模型无法输出生成结构对应数据库中的哪个分子（CID） |
| 4 | 评估指标不够全面：仅使用 RMSD，无法反映化学有效性等关键维度 |
| 5 | 未利用环结构刚性先验：五元环和六元环等固定结构未被利用以降低生成复杂度 |
| 6 | 未利用其他分子物理结构不变性：键角、平面性等化学规律未被用于约束生成 |
| 7 | 底部原子位置准确性不足：被遮挡区域的原子重建精度偏低 |

---

## 二、总体改进架构

### 2.1 改进前架构

原始流程为：AFM 图像堆栈经 Video ViT 编码器提取条件向量，再由条件扩散模型逐原子生成全部坐标和类型。模型将原子数作为外部输入，没有物理约束，也没有分子识别能力。

### 2.2 改进后架构

改进后的推理流程如下：

1. AFM 图像堆栈输入 Video ViT 编码器，输出 512 维条件向量
2. 条件向量同时送入三个分支：
   - **原子数预测头**：预测分子包含的原子数量 N
   - **分子对比检索头**：通过对比学习检索数据库中最相似的分子 CID（Top-5）
   - **条件扩散模型**：以预测的 N 构建掩码，生成三维坐标和原子类型
3. 扩散去噪过程中，融入**环结构刚体约束投影**，逐步将环内原子拉回标准构型
4. 生成完成后，经 **RDKit 分子力场弛豫**后处理，修正微小畸变

> **改进要点**：从"单一生成"升级为"预测+生成+约束+后处理"的四层架构，每一层分别解决不同问题。
>
> **改进优点**：原子数不再依赖外部输入（解决问题1）；生成过程和后处理双重保障物理合理性（解决问题2、6）；新增分子检索能力（解决问题3）；环约束降低生成复杂度（解决问题5）。

## 三、新增模块详解

### 3.1 原子数预测头（AtomCountHead）

**对应问题**：问题1——生成的原子个数不正确

**改进方式**：在 ViT 编码器的 CLS token 输出之后，新增一个独立的原子数预测头，采用"分类+回归"双分支设计。分类分支将原子数离散化为 1~85 个类别进行预测；回归分支直接输出一个连续值。推理时以分类分支的 argmax 作为最终预测原子数，用于构建扩散模型的生成掩码。

训练损失由两部分组成：分类分支使用交叉熵损失，回归分支使用 SmoothL1 损失，两者加权求和。

> **改进优点**：
> - 模型在推理时不再需要真实原子数，实现了端到端的自主预测
> - 双分支设计兼顾了分类的离散精度和回归的梯度平滑性
> - 预测头直接复用 ViT 编码器的特征，不增加额外编码开销

### 3.2 分子对比检索头（MoleculeRetrievalHead）

**对应问题**：问题3——无法识别分子 ID

**改进方式**：放弃传统的直接分类方法（当分子 CID 数量很大时泛化性差），改用**对比学习 + 余弦相似度检索**。

具体设计为：ViT 的 CLS token 经过一个投影网络映射到 128 维的嵌入空间并做 L2 归一化。同时为数据库中的每个已知分子维护一个可学习的嵌入向量。训练时采用 InfoNCE 对比损失：计算查询嵌入与所有分子嵌入的余弦相似度，除以温度系数后作为 logits 进行交叉熵优化。推理时直接检索相似度最高的 Top-5 候选分子。

> **改进优点**：
> - 对比学习天然适合检索任务，嵌入空间中语义相近的分子距离更近
> - 对未在训练集中出现过的分子也具有一定的泛化能力
> - Top-5 检索比 Top-1 分类具有更高的容错率

### 3.3 环检测与刚体词表模块

**对应问题**：问题5——未利用环结构刚性先验

**改进方式**：在数据预处理阶段引入环检测模块。该模块不涉及神经网络，是纯算法模块。

**环检测流程**如下：

1. **构建分子图**：基于原子类型对应的标准键长查找表（如 C-C 键 1.54 Å、C-H 键 1.09 Å 等），对所有原子对计算欧氏距离，距离在键长的 1.2 倍以内则判定为成键，建立邻接矩阵
2. **环搜索**：使用 BFS/DFS 算法在分子图中搜索所有长度为 5 和 6 的简单环（最小环集 SSSR），去除重复
3. **环分类**：根据环内原子的元素组成分类：全碳六元环为苯环，含一个氮的六元环为吡啶，含一个氧的五元环为呋喃等
4. **刚体参数化**：对每个检测到的环计算其质心坐标、最佳拟合平面的法向量（编码倾角信息），以及从标准模板到实际位置的旋转矩阵

**刚体词表**（Ring Templates）预定义了每种环类型的标准局部坐标。例如苯环为正六边形，芳香碳碳键长 1.40 Å 对应的归一化半径为 0.117。

> **改进优点**：
> - 一个苯环（6 个原子）原需 6×4=24 个自由度，参数化后仅需 7 个（中心 3 + 法向量 3 + 类型 1），生成复杂度降低约 70%
> - 环结构几何高度固定，利用刚体先验可消除环内原子的畸变
> - 预处理阶段完成，不增加训练时的计算开销

### 3.4 物理约束模块（Physics-Informed Constraints）

**对应问题**：问题2——物理结构不合理；问题6——未利用其他结构不变性

**改进方式**：设计三种可微分的物理惩罚损失函数，在训练过程中作为软约束加入：

1. **键长约束**：对预测坐标中距离在成键范围内的原子对，计算其距离与理想键长的偏差。当偏差超过容差（约 0.15 Å）时施加二次惩罚。理想键长来自化学键长查找表（如 C-C 单键 1.54 Å、C=C 双键 1.34 Å、C-H 键 1.09 Å 等）

2. **键角约束**：对成键的三原子组合计算键角，与理想键角比较。sp3 杂化碳约 109.5 度，sp2 杂化碳（芳香环）约 120 度，以余弦值偏差的平方作为惩罚

3. **平面性约束**：对芳香环及 sp2 体系中的原子，计算其偏离最佳拟合平面的程度，以离面偏差的平方作为惩罚

> **改进优点**：
> - 软约束以损失函数形式融入训练，梯度可反向传播，不会阻断训练
> - 三种约束分别从键长、键角、共面性三个维度确保化学合理性
> - 所有常量在归一化坐标空间中定义（除以 12.0），与模型坐标空间一致

### 3.5 RDKit 力场后处理弛豫

**对应问题**：问题2——物理结构不合理（推理时兜底修正）

**改进方式**：在推理时增加一个后处理步骤。将生成的原子坐标和类型转换为 RDKit 的分子对象，然后使用 MMFF94 分子力场进行能量最小化优化（最多 200 步迭代），将微小的键长/键角畸变"拉回"物理合理区间。

设置弛豫幅度上限：单个原子的位移不超过 0.3 Å，防止大幅偏移。若 MMFF94 无法参数化，自动回退到 UFF 力场。此步骤仅在推理时使用，不参与训练。

> **改进优点**：
> - 作为训练时软约束的补充，在推理时提供"硬性兜底"
> - MMFF94 是成熟的化学力场，对有机分子的几何优化效果成熟可靠
> - 位移上限控制确保不会破坏模型已经正确预测的结构部分

## 四、现有模块改进

### 4.1 数据集模块改进

**改进内容**：

1. **返回分子 CID 索引**：在数据加载时返回每个样本对应的 CID 标识及其在训练集中的数值索引，用于对比检索损失的计算。训练集先建立 CID 到索引的映射，验证集和测试集复用该映射

2. **返回环检测预处理结果**：在加载坐标后调用环检测模块，将检测到的环中心、法向量、类型、模板坐标等信息一并返回，供扩散模型的环约束使用

> **改进优点**：CID 信息贯穿整个管线，使分子检索训练成为可能；环信息在预处理阶段一次性计算，训练时直接读取不增加开销

### 4.2 扩散模型改进

**对应问题**：问题5（环约束）、问题7（底部原子精度）

**改进内容**：

1. **Z 轴深度感知加权损失**：在计算坐标均方误差时，不再对所有原子平等对待。根据每个原子在真实结构中的 Z 坐标，底层（被遮挡）原子获得 3 倍权重，顶层原子保持 1 倍权重。权重按 Z 坐标线性插值。这迫使模型将更多的学习能力投入到推断看不见的底部结构

2. **环一致性损失**：在训练时，从噪声预测反推出预估的干净坐标，检查其中环结构的保持程度，作为额外损失项

3. **去噪过程中的环约束投影**：在反向采样的每一步 DDPM 更新后，对检测到的环结构执行刚体约束投影。具体做法是：提取环内原子的当前位置，通过 Procrustes 对齐（基于 SVD 分解）找到标准模板到当前位置的最优刚性变换，然后将环原子位置向理想刚性位置混合。混合系数按去噪进度调度：前期弱约束（避免限制探索），后期强约束（确保最终结构合规）

4. **推理时无真实环信息的处理**：在去噪到第 700、500、300、100 步时，对中间结构自动运行环检测器，用新检测到的环信息指导后续的约束投影

> **改进优点**：
> - Z 轴加权直接解决底部原子精度不足的核心问题（问题7）
> - 环约束投影确保生成的环结构始终保持正确几何形状
> - 渐进式混合调度平衡了生成自由度和约束严格度
> - 推理时的自适应环检测使得模型不依赖真实标注

### 4.3 训练流程改进

**改进内容**：

1. **模型结构扩展**：在原有的 ViT 编码器 + 扩散模型基础上，新增原子数预测头、对比检索头和分子嵌入库三个组件

2. **生成流程改造**：推理时先由原子数预测头预测 N，再由扩散模型以 N 构建掩码进行生成，同时对比检索头输出 Top-5 候选 CID，最后经 RDKit 力场弛豫输出最终结构

3. **三阶段训练策略**：

| 阶段 | 训练轮次 | 损失组成 | 学习率 | 目标 |
|------|---------|---------|--------|------|
| **阶段一：基础训练** | 第 1~30 轮 | 坐标噪声预测损失 + 原子类型损失 + 原子数预测损失 + 对比检索损失 | 1e-4 余弦退火至 1e-6 | 学会基本的结构重建和辅助预测 |
| **阶段二：约束训练** | 第 31~45 轮 | 阶段一全部损失 + 键长约束损失 + 环一致性损失 | 5e-5 余弦退火至 1e-6 | 提升生成结构的物理有效性 |
| **阶段三：底部聚焦** | 第 46~60 轮 | 阶段二全部损失 + 底部原子 3 倍加权 | 2e-5 余弦退火至 1e-6 | 专门提升遮挡区域原子的重建精度 |

> **改进优点**：
> - 三阶段策略避免了在训练早期引入复杂约束导致的优化困难
> - 每个阶段有明确的优化目标，损失权重渐进式引入
> - 最终阶段专注底部原子，与论文核心贡献（遮挡补全）高度对齐

## 五、评估指标改进

**对应问题**：问题4——RMSD 不够全面

### 5.1 新增指标

| 指标名称 | 说明 | 解决的问题 |
|---------|------|-----------|
| **原子数准确率** | 预测原子数与真实值的精确匹配率及平均绝对误差 | 问题1 |
| **键有效率** | 检测到的化学键中，键长在合理范围内的比例 | 问题2 |
| **环结构保持分数** | 正确重建的环数占真实环总数的比例 | 问题5 |
| **CID 检索准确率** | Top-1 和 Top-5 命中率 | 问题3 |
| **底部原子 RMSD** | 仅针对底部 30% 原子计算的 RMSD | 问题7 |

### 5.2 综合评分公式

设计一个加权综合评分，将所有子指标归一化到 [0, 1] 区间后加权求和：

| 子指标 | 权重 | 含义 |
|--------|------|------|
| 坐标精度（1 - RMSD/最大值） | 0.30 | 整体三维重建精度 |
| 底部原子评分 | 0.20 | 遮挡区域重建能力（本文核心关注点） |
| 键有效率 | 0.15 | 化学键的物理合理性 |
| 环结构保持分数 | 0.15 | 关键子结构的完整性 |
| 原子数准确率 | 0.10 | 分子规模预测能力 |
| CID 检索准确率 | 0.10 | 分子识别能力 |

> **改进优点**：
> - 从单一 RMSD 扩展到 6 维度评估，全面反映模型能力
> - 底部原子和坐标精度权重最高（共 0.50），体现论文的遮挡补全核心目标
> - 综合评分提供单一可比较的数值，便于不同方法的横向对比

## 六、关键技术细节

### 6.1 坐标空间约定

所有模型操作在归一化坐标空间中进行：真实坐标除以 12.0（半个 24 Å 晶胞尺寸），映射到 [-1, 1] 范围。因此：

- C-C 单键 1.54 Å 在模型空间中为 0.128
- C-H 键 1.09 Å 为 0.091
- 键长容差 0.15 Å 为 0.0125
- 苯环半径 1.40 Å 为 0.117

### 6.2 Procrustes 对齐算法

环约束投影的核心算法。给定模板点集和目标点集（均已中心化），通过 SVD 分解找到最优的旋转矩阵，使模板经旋转后与目标最小二乘对齐。该算法可微分，支持梯度反向传播。需要特别处理反射情况（当旋转矩阵行列式为负时翻转符号）。

### 6.3 推理完整流程

```
输入：AFM 10 层 Z-slice 图像堆栈

Step 1: Video ViT 编码 → 512 维条件向量

Step 2: 三路并行预测
  (a) 原子数预测头 → 预测 N 个原子
  (b) 对比检索头 → Top-5 候选分子 CID
  (c) 条件扩散模型 → 1000 步去噪生成

Step 3: 去噪过程中的环约束
  在第 200~1000 步，每步执行 Procrustes 刚体投影
  在第 300、500、700、900 步自动检测环结构

Step 4: RDKit MMFF94 力场弛豫（后处理）

输出：三维原子坐标 + 原子类型 + 预测原子数 + 候选分子 CID
```

## 八、改进总结

| 问题 | 改进措施 | 核心优点 |
|------|---------|---------|
| 1. 原子数不正确 | 独立预测头（分类+回归双分支） | 端到端自主预测，不依赖外部输入 |
| 2. 物理结构不合理 | 训练时软约束 + 推理时 RDKit 力场弛豫 | 双重保障，训练优化+推理兜底 |
| 3. 无法识别分子 ID | 对比学习检索头 | 泛化性强，支持 Top-K 容错检索 |
| 4. 评估指标单一 | 6 维综合评分体系 | 全面反映化学有效性和重建精度 |
| 5. 未利用环结构先验 | 刚体词表 + 去噪环约束投影 | 生成复杂度降低 70%，环结构始终合规 |
| 6. 未利用结构不变性 | 键长/键角/平面性三重约束 | 从多维度确保化学合理性 |
| 7. 底部原子精度不足 | Z 轴深度感知加权 + 三阶段训练聚焦 | 直接倒逼模型重点学习遮挡区域 |
