## 六、核心判断
- 这个 EXP-02 使用的是 test split 闭集检索协议，因此它最适合回答“当前对象级恢复能否支持候选缩小”。
- `Top1` 衡量直接命中能力，`Top3/Top5` 更接近当前论文的 candidate set reduction 叙事。
- 分层表应重点观察：复杂样本上检索下降是否主要伴随 `pred_object_type_acc`、`pred_object_edge_f1` 或 `pred_object_z_mae` 恶化。
- 如果高 `pred_object_score` / 低 `pred_object_z_mae` 分层明显更强，说明对象级结构重建和检索能力是同向变化的。
