# V19 训练结果中文对比报告：v19_object_joint_full6h vs v19_object_joint_full15_all

## 一、对比对象
- 基线实验：`v19_object_joint_full6h`
- 当前实验：`v19_object_joint_full15_all`
- 基线最佳轮次：`12`
- 当前最佳轮次：`15`

## 二、关键结论
- `gt_object_score - peak_object_score` 的差距从 `0.2306` 缩小到 `0.0296`，字段名 `gap_delta` = `-0.2010`。这说明预测中心闭环能力明显更接近真实中心上限。
- 字段名 `peak_object_score`：`0.7171` -> `0.8016`，说明整体对象级闭环重建继续提升。
- 字段名 `peak_center_type_acc`：`0.7257` -> `0.8185`；字段名 `peak_center_macro_f1`：`0.5623` -> `0.6564`。说明预测中心下的原子类型能力明显变强。
- 字段名 `peak_center_hetero_f1`：`0.5674` -> `0.8649`。说明杂原子识别是这轮提升最明显的方向之一。
- 字段名 `peak_center_edge_f1`：`0.7981` -> `0.8138`，字段名 `atom_z_mae_r3`：`0.0987` -> `0.0893`。说明边恢复和 z 轴误差也有改善，但幅度小于类型和杂原子。

## 四、总体判断
- 当前这轮 `full15_all` 相比 `full6h`，不是单项偶然波动，而是对象级闭环能力整体更强。
- 最大的实质提升来自：预测中心下的类型、杂原子，以及真实中心上限与预测中心闭环之间差距的缩小。
- 稠密二维类型图仍然偏弱，后续仍应优先发展对象级类型/边头，而不是重新把主希望压回稠密 `type map`。
